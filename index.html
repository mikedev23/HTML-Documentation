Introduction

HTML stands for Hypertext Markup Language.

It is the standard markup language for creating web pages and web applications.

Structure

The structure of an HTML document consists of different elements.

These elements define the building blocks of a webpage and organize the content.

Head

The <head> element is used to provide information about the webpage.

It contains meta information, CSS stylesheets, JavaScript scripts, and more.

Body

The <body> element contains the visible content of the webpage.

It includes text, images, links, and other elements that are displayed to the user.

  1. Start with an opening <body> tag.
  2. Add the desired content inside the opening and closing tags.
  3. End with a closing </body> tag.

Hello World

To display "Hello, World!" in HTML, you can use the following code:

<p>Hello, World!</p>

This will render the text "Hello, World!" as a paragraph on the webpage.

  1. Start with an opening <p> tag.
  2. Write the desired text inside the opening and closing tags.
  3. End with a closing </p> tag.
